=====================

TECA is a framework for parallel data analytics on large scale systems built by
the DOE. TECA has algorithms for extreme event detection and post detection
analysis.  Written in C++, TECA uses MPI + X , where X is one of threads,
OpenMP, or GPUs, to deliver state of the art performance and scaling. It's
Python bindings provide an easy way to script analysis suitable to any need and
to develop new diagnostics.
